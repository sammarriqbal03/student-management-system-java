# Student Management System (Java OOP)

## 📌 Description

This is a console-based Student Management System built using Java and Object-Oriented Programming concepts. It manages both students and teachers efficiently.

## 🎯 Features

* Add Student
* View Students
* Search Student
* Delete Student
* Add Teacher
* View Teachers
* Separate service classes for better structure
## 🧱 OOP Concepts Used
* Classes & Objects
* Inheritance (Person → Student, Teacher)
* Encapsulation
* Separation of concerns (Service classes)

## 🛠️ Technologies Used
* Java
* IntelliJ IDEA

## ▶️ How to Run
1. Open project in IntelliJ
2. Run `Main.java`
3. Follow menu options in console
##  Project Structure

* Person.java
* Student.java
* Teacher.java
* StudentService.java
* TeacherService.java
* Main.java
